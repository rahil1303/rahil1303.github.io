  1. OBFI: Implementation, Empirical Analysis, and Crossover Characterization of Competing Protocols
    Rahil Sharma
    CWI Amsterdam, 2026
    MSc thesis, VU Amsterdam / UvA. Supervised by Dr. M. van Dijk, CWI Amsterdam. Manuscript in preparation.
    Abs Code

    This thesis introduces and implements Oblivious Bloom Filter Insertion (OBFI), a protocol for constructing encrypted Bloom filters while concealing insertion access patterns from a semi-honest server. We present the first empirical implementation and systematic comparison of OBFI against an alternative privacy-preserving protocol (WS12) for the same problem. Using a distributed Python system built on gRPC, we run a sensitivity study across varying batch sizes and filter capacities, characterising the crossover point at which one protocol becomes preferable to the other and developing a deployment decision framework based on observed performance trends.

  2. Auditable Financial Fraud Detection: Graph Features and Agentic Investigation
    Rahil Sharma
    2026
    VU Amsterdam / University of Amsterdam. Preprint in preparation.
    Abs arXiv Code

    We present an end-to-end fraud detection pipeline built on 6.3M+ financial transactions, designed around two principles: detections must be explainable, and the system must know when to defer to human judgement. We engineer graph-based and anomaly-detection features from transaction network structure, validate the graph-based approach through a controlled fraud-ring experiment, and build an AI-assisted investigation workflow that combines model explanations, transaction context, and similar historical cases into structured case rationales. A confidence-based escalation policy automatically resolves clear cases and routes conflicting decisions to human review, maintaining a full audit trail throughout.

  3. Adaptive AI Interaction and Reasoning Platform: Multi-Model Evaluation of RAG Architectures
    Rahil Sharma
    2026
    VU Amsterdam / University of Amsterdam. Preprint in preparation.
    Abs arXiv Code

    We build a full-stack Retrieval-Augmented Generation platform supporting multi-model comparison across LLaMA, Qwen, GPT, and Groq-hosted model families. The platform treats LLM evaluation as a systems problem: retrieval configuration, model selection, and evaluation criteria are independently controllable, enabling isolation of individual design choices on system behaviour. We develop semantic chunking and vector retrieval pipelines with configurable similarity metrics, multi-round critique and evaluation pipelines, and exportable evaluation logs for reproducible comparison across model families.

  1. Energy–Performance Trade-offs of Java Garbage Collection Strategies
    Rahil Sharma
    2025
    VU Amsterdam / University of Amsterdam. Preprint in preparation.
    Abs arXiv Code

    We conduct a quantitative empirical study comparing Serial, Parallel, and G1 garbage collectors under controlled Java workloads, investigating trade-offs between energy consumption, throughput, and latency. Measurements use EnergiBridge and Intel RAPL for hardware-level energy profiling. Using a Randomized Complete Block Design experimental setup with automated execution pipelines, we apply ANOVA and Tukey HSD statistical methods to evaluate significance of differences across configurations and compare OpenJDK and Oracle JDK implementations under varying workload conditions.

  2. Reactivity–Reflectivity Vector Framework: Computational Analysis of Political Polarization and Media Framing
    Rahil Sharma
    2025
    VU Amsterdam / University of Amsterdam. Preprint in preparation.
    Abs arXiv Code

    We design a computational framework for studying political polarization, media framing, and audience reactivity using large-scale YouTube transcript and comment data. The framework operationalizes qualitative concepts — framing, reflectiveness, audience reactivity — into measurable computational features, enabling structured cross-media and cross-topic comparisons. We collect and process data across major political events and news outlets, apply NLP techniques to translate behavioural and engagement patterns into quantitative features, and combine computational analysis with qualitative interpretation to study how narrative structure shapes audience response.

  1. Explainable AI Framework for Privacy-Aware Feature Attribution
    Rahil Sharma
    2024
    VU Amsterdam / University of Amsterdam. Preprint in preparation.
    Abs arXiv Code

    We develop an automated framework for privacy-aware feature selection aligned with GDPR data minimization principles. The framework combines SHAP values with knowledge graphs to provide interpretable, context-aware feature attribution, producing selections that are auditable at the relationship level. We implement PAC-privacy and autoencoder-based encoding mechanisms for secure data handling and build an end-to-end pipeline supporting heterogeneous datasets across numerical and categorical features.
