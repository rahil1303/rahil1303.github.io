TA for **Text Mining for AI** at VU Amsterdam. Assisted students in understanding core text mining techniques including natural language processing, information retrieval, and machine learning applications for text data.

Led lab sessions, guided project work, and provided feedback on assignments. Topics covered text classification, named entity recognition, semantic similarity, and practical NLP pipelines.
